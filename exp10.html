Study of XML

What is XML?

XML stands for Extensible Markup Language. It is used to store and transport data. XML is self-descriptive and designed to carry data, not to display data.

Features of XML

  • Easy to understand
  • Platform independent
  • Stores and transports data
  • User-defined tags
  • Readable by both humans and machines

Advantages of XML

  • Simple data sharing
  • Supports different applications
  • Easy data transfer over internet
  • Flexible structure

Sample XML Code

<?xml version="1.0"?>

<student>
    <name>Sejal</name>
    <branch>CSE</branch>
    <year>Second Year</year>
</student>

Applications of XML

  • Web services
  • Data storage
  • Configuration files
  • Information sharing

Conclusion

XML is an important markup language used for storing, transporting, and organizing data efficiently between systems.
