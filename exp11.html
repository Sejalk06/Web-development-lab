XSLT Transformation to XML

XSLT (Extensible Stylesheet Language Transformations) is used to transform XML documents into another format like HTML or another XML document.

XML File (student.xml)

<?xml version="1.0"?>

<students>
    <student>
        <name>Sejal</name>
        <branch>CSE</branch>
        <year>SY</year>
    </student>

    <student>
        <name>Rahul</name>
        <branch>IT</branch>
        <year>TY</year>
    </student>
</students>

XSLT File (student.xsl)

<?xml version="1.0"?>

<xsl:stylesheet version="1.0"
xmlns:xsl="http://www.w3.org/1999/XSL/Transform">

<xsl:template match="/">

<html>
<body>

<h2>Student Information</h2>

<table border="1" cellpadding="5">

    <tr bgcolor="lightblue">
        <th>Name</th>
        <th>Branch</th>
        <th>Year</th>
    </tr>

    <xsl:for-each select="students/student">

    <tr>
        <td>
            <xsl:value-of select="name"/>
        </td>

        <td>
            <xsl:value-of select="branch"/>
        </td>

        <td>
            <xsl:value-of select="year"/>
        </td>
    </tr>

    </xsl:for-each>

</table>

</body>
</html>

</xsl:template>

</xsl:stylesheet>

Output

The XML data will be transformed into an HTML table using the XSLT stylesheet.
